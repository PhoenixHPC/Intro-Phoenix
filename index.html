About Hand-out Style

This repository contains style files that can be used by LaTeX to generate consistently styled handout documents for use in hands-on training workshops.

If you are interested, full details of the templates used for this workshop can be find in style repository at: http://BPA-CSIRO-Workshops.github.io/handout-template/

Credits

This style was originally designed for the Next Generation Sequencing (NGS) hands-on workshop developed by Bioplatforms Australia (BPA), CSIRO Bioinformatics Core and the EBI. It was crafted predominantly by Dr. Nathan S. Watson-Haigh.
